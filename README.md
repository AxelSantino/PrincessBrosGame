# Vertical Platformer - 2D Java Game
Este proyecto es un videojuego de plataformas de progresión vertical desarrollado completamente en Java, sin el uso de motores gráficos externos. El principal desafío y diferencial técnico del proyecto es la implementación manual del motor de físicas y la lógica espacial. El código gestiona de forma nativa la gravedad, la detección de colisiones precisas (hitboxes contra techos, plataformas y límites de pantalla) y la interacción en tiempo real entre el personaje principal y los distintos objetos del entorno. Es una demostración práctica de algoritmia aplicada, manejo de estados y renderizado por software.
